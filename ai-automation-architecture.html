How It Works

  • • User sends input via Telegram, Email, Web, or Sheets
  • • Hermes Agent receives and analyzes the request
  • • Orchestrator delegates to specialized sub-agents
  • • AI model handles reasoning (your choice of provider)
  • • Result delivered back — report, data update, or alert

Key Features

  • • Multi-agent orchestration (parallel processing)
  • • Self-healing watchdog — auto-restart on failure
  • • Cron scheduler — runs 24/7 on autopilot
  • • Multi-platform delivery — Telegram, Email, Sheets
  • • Model-agnostic — OpenAI, Anthropic, or local Ollama

Use Cases

  • • Automated market research reports
  • • Voice agent for order taking
